## CQ-editor

The fastest way to try CadQuery is with CQ-editor, a graphical editor for CadQuery scripts. Packages are provided below for easy installation - for other installation methods, see the [CadQuery documentation](https://cadquery.readthedocs.io/en/latest/installation.html).

|----------|----------|

| macOS (ARM64) | [CQ-editor-macos-arm64.zip](https://github.com/CadQuery/CQ-editor/releases/download/0.7.0/CQ-editor-macos-arm64.zip) |
| Windows (x86_64) | [CQ-editor-windows-x86_64.exe.zip](https://github.com/CadQuery/CQ-editor/releases/download/0.7.0/CQ-editor-windows-x86_64.exe.zip) |
